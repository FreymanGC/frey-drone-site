# Crea un sitio web profesional,

Aplicación generada con [Darwin Now Magic](https://magic.darwinnow.io)

## Comenzar

```bash
# Instalar dependencias
npm install

# Ejecutar en modo desarrollo
npm run dev

# Construir para producción
npm run build

# Vista previa de producción
npm run preview
```

## Tecnologías

- React 18
- Vite
- Tailwind CSS
- Font Awesome 6 Free Icons
